# 🧮 Simple Python Calculator

## 📝 Project Overview
This is a beginner-friendly command-line calculator built using **Python**. I developed this project to practice fundamental programming concepts like functions, loops, and error handling (try-except blocks).

## 🚀 Features
- **Basic Arithmetic:** Supports addition, subtraction, multiplication, and division.
- **Error Handling:** Prevents the program from crashing if a user tries to divide by zero or enters non-numeric values.
- **User-Friendly Interface:** Clear instructions in the terminal for an easy experience.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Tools:** GitHub for version control

## 📖 What I Learned
Through this project, I strengthened my understanding of:
1. Defining and calling functions in Python.
2. Managing user input and data types.
3. Implementing logical conditions for a smooth program flow.
